# MediSlot

MediSlot is a full-stack healthcare appointment scheduling platform built using Flask and MySQL.

## Features

- Authentication & Role-Based Access
- Super Admin Panel
- Doctor Approval Workflow
- Appointment Booking
- Time Slot Scheduling
- Appointment Approval/Rejection
- Doctor Availability Management
- Responsive Dashboard UI

## Tech Stack

- Flask
- Python
- MySQL
- SQLAlchemy
- HTML
- CSS
- Bootstrap
- Jinja2

## Setup

1. Clone repository
2. Install requirements
3. Configure .env
4. Run Flask app
